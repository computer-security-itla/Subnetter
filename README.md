# Installation:
```
cd Subnetter
python subnetter.py
```

# Usage:
Commands

```
To show available commands, use /help
To exit the program, use /exit
Commands like /set and /subnet can have arguments.
Examples:
/set 192.168.10.0
/set 192.168.10.0/24
/subnet 8

```
