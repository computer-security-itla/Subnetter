# Installation:
```
cd Subnetter
python3 subnetter.py
```

# Usage:
Commands

```
To show available commands, use /help
To exit the program, use /exit
Commands like /set and /subnet have optional arguments.
Examples:
/set 192.168.10.0
/set 192.168.10.0/24
/subnet 8
Commands like /name have required arguments
Example:
/name EDIF-A
/name LAN-1

```
